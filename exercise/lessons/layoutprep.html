Layout Preperations

You'll need to setup your manual with some additional "div" tags to begine the next steps which are CSS (Cascading Style Sheets) and JS (Javascript).

To begin, you'll need to wrap all the code you wrote "div" tags to help segment and organize your code.

Step 1

Wrap all the code within the "body" tag with a "div" tag with the "id" attribute set to "content_column".

Step 2

Before this new div you need to make a place (in the code) for a navigation bar.

Create a "div" with an "id" attribute set to "nav_column" making sure it is before the "content_column".

Step 3

Now you need to wrap all the code inside the "body" tag again with another "div" tag.

Wrap all the code within the "body" tag with a "div" tag with the "id" attribute set to "content".

You have just one more "div" to go.

Step 4

Now you need to wrap all the code inside the "body" tag again with another "div" tag.

Wrap all the code within the "body" tag with a "div" tag with the "id" attribute set to "page_container".

I know all these "div" tags seem silly now, but it will make your coding moving forward much easier. You'll see why when you move to CSS and JS.

Once you've completed these steps, refresh the web page.
