Links, Lists and Tables (Part 3)

This is the last part of a multipart exercise that will deal with working on the same piece of HTML code. As always, just follow the instructions to complete this exercise.

HTML tables allow you to arrange data -- text, images, links, lists, etc. -- into rows and columns of cells.

Step 1

Around your newly created lists, you are going to wrap a few different tags. First wrap each list with "td" (cell) tags. There should be two cells (one for each of your lists).

Step 2

Wrap both lists and "td" tags in "tr" (row) tags. There will currently only be a single row in your table so there should only be a single open and a single closed "tr" tags.

Step 3

Wrap your new table row in "table" tags. Your two lists will be side by side now instead of one on top of the other.

Optional Step 4

Add an additional row before the row that holds your two lists. Instead of using "td" tags, use "th" (table header) tags to be used as cells and fill in two "th" tags with "Unordered List" and "Ordered List". This will give your two table cells headers explaining the content below them.

Once you've completed these steps, refresh the web page.
