Divs

Step 1

Create a new header using h1 tags and input "Divs and Spans".

Step 2

Create a "div" tag with an "id" attribute set to "first_div".

"div"s are what's called a block level element meaning that it's width is equal to it's parent's width.

To demonstrate, the "div" you created will have it's background color changed to red.

Once you've completed these steps, refresh the web page.
