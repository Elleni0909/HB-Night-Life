Forms (Part 2)

This is part of a multipart exercise that will deal with working on the same piece of HTML code. As always, just follow the instructions to complete this exercise.

In this exercise, we're going to be adding a couple new form elements. Textarea form elements are for inputting large, multi-line amounts of text. Select/Option elements are for creating dropdown menu boxes.

Step 1

Staying inside of the form tags that you made in the last exercise and after your input box, append a self closing "br" (line break) tag followed by a new "textarea" tag. Unlike the "input" tag, "textarea" needs to have it's own closing tag.

Step 2

Add two attributes to your "textarea" tag. The first attribute is "name" and its value is "bio". The second attribue is "id" and its value is also "bio".

Step 3

Let's give the input box and text area some labels so we can keep track of them before continuing.

Add a "label" tag before your "input" tag and give it the "for" attribute and the value for that attribute should be set to "name". Set the data between the opening and closing "label" tags to "Name:".

Add another "label" tag before your "textarea" tag and give it the "for" attribute and the value for that attribute should be set to "bio". Set the data between the opening and closing "label" tags to "Bio:".

Step 4

Now back to the form elements. Add another self-closing "br" tag after your text area and create a new "select" tag. The two attributes to be added to the the "select" tag are, again, "name" and "id". Create the two attributes and give them both the value of "month".

Step 5

Inside of the "select" tag, add 12 (yes, 12) "option" tags. Each of these "option" tags will correspond to a month of the year. Create an attribute called "value" for each "option" tag and set "value" attributes to "January" through "December".

Input text between the opening and closing "option" tags equal to the months as well.

Step 6

Add another "label" tag before your "select" tag and give it the "for" attribute and the value for that attribute should be set to "month". Set the data between the opening and closing "label" tags to "Birth Month:".

Step 7

Add a self-closing "br" tag after the closing "select" tag.

Once you've completed these steps, refresh the web page.
