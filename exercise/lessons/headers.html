Headers exercise

Follow the instructions below to complete the exercise.

Preamble

In this exercise, you will be editing the index.html file.

Also known as header tags or "H" tags. Header tags are used to denote header text for sections of your web site.

The main title of your website is denoted with the "h1" tag. Currently that is set to "My Manual" and has been added to the site automatically for you.

Step 1

Create an "h1" header within the body of your page. (Remember, header tags need both an opening and a closing tag.)

Step 2

Input "My First Header" between your openeing and closing "h1" tags.

Optional Step 3

Add additional headers under your h1 tags using the h2, h3, h4, h5, and h6 tags.

Once you've completed these steps, refresh the web page.
