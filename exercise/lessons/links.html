Links, Lists and Tables

This is a multipart exercise that will deal with working on the same piece of HTML code. As always, just follow the instructions to complete this exercise.

Links (also known as anchor tags, hyperlinks or "A" tags) are a way websites get from place to place.

Step 1

Create a new header using h1 tags and input "Links, Lists and Tables".

Step 2

Create an "a" link after the header you just created. (Remember, "a" tags need both an opening and a closing tag.)

Step 3

Input the words "Hackbright Academy" between your new "a" tag.

Step 4

We're going to do something different to this "a" tag that we haven't done to previous tags. We're going to add an attribute to the tag. Specifically, we need to add the "href" (Hyperlink Reference) attribute to the opening "a" tag.

To add the "href" attribute, input a space after the letter "a" in your opening tag followed by href="".

Finally, put "http://www.hackbrightacademy.com" between the "s in the new "href" attribute.

Optional Step 5

Add more links to some of your favorite sites using "a" tags. (Don't forget the "href" attribute.)

Once you've completed these steps, refresh the web page.
